# Simple Calculator

This is a basic calculator web application that performs standard arithmetic operations like addition, subtraction, multiplication, and division.  
It provides a clean and interactive user interface and works directly in the browser.

## Tech Stack
- HTML
- CSS
- JavaScript

## Features
- Button-based number and operator input
- Real-time display of calculations
- Supports +, −, ×, ÷ operations
- Clear and reset functionality

## How to Use
1. Open `index.html` in any web browser.
2. Click the buttons to enter numbers and operations.
3. Press `=` to get the result.
4. Use `C` to clear the display.

This project is a simple demonstration of DOM manipulation and event handling in JavaScript.

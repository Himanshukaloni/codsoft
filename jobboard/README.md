# Job Portal Web Application

This is a full-stack job portal where recruiters can post jobs and applicants can register and apply.  
It manages user accounts, job listings, and applications with data stored in a MongoDB database.  
The system connects a dynamic frontend with a Node.js and Express backend for all operations.

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- Database: MongoDB

## Main Features
- User registration and login
- Recruiter dashboard to add and manage jobs
- Applicant dashboard to view and apply for jobs
- Store and fetch job and applicant data from database
- Secure routing and backend API handling

## Purpose
This project demonstrates a complete job management system with authentication, database integration, and role-based task handling using a full-stack JavaScript backend.

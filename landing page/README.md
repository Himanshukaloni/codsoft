# Landing Page Website

This is a simple and responsive landing page built using HTML and CSS.  
It presents a clean layout with sections for introduction, content, and basic styling.  
The page is fully static and runs directly in the browser without any backend.

## Tech Stack
- HTML
- CSS

## Features
- Responsive and modern layout
- Structured sections for headline and content
- Custom styling using pure CSS
- Lightweight and fast loading

## Purpose
This project demonstrates basic webpage design and layout creation using only HTML and CSS.

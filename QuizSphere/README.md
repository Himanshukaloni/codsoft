QuizSphere is a full-stack web application for creating and solving quizzes.  
Users can add new quizzes, attempt existing quizzes, and view results.  
Quiz data is stored in a MongoDB database through a Node.js and Express backend.

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- Database: MongoDB

## Main Features
- Create and manage quizzes
- Solve quizzes with instant result calculation
- Store and retrieve quiz data from database
- REST API based communication between frontend and backend

## Purpose
This project demonstrates a complete quiz system with database integration and dynamic task handling using the MERN-style backend (without React).
